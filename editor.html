        Setup

        G.init({cols, rows, tileSize, fps})

        G.tiles = { '#': {bg, fg, ch, passable} }

        G.map = array of strings (rows)


        Callbacks

        G.onKey = (key) => {}

        G.onUpdate = () => {}

        G.onDraw = (ctx) => {}


        Grid helpers

        G.getCell(x, y) → char

        G.setCell(x, y, ch)

        G.isPassable(x, y) → bool

        G.inBounds(x, y) → bool

        G.findCell(ch) → {x,y} | null

        G.countCell(ch) → number

        G.randomEmpty() → {x,y}

        G.dirFromKey(key) → {dx,dy} | null

        G.borderedMap(cols, rows) → string[]


        Draw helpers (in onDraw)

        G.text(x, y, str, color, size)

        G.fillTile(tx, ty, color)

        G.state — free object for game data

        G.start() / G.stop()


        Sound

        G.beep(freq, dur, type, vol)

        freq Hz · dur seconds · type: 'square' 'sine' 'sawtooth' · vol 0–1
